# Store Image Placeholders

Add store images in this tree when preparing release metadata.

Recommended paths:

- `phoneScreenshots/` for mobile screenshots
- `sevenInchScreenshots/` for 7-inch tablet screenshots (optional)
- `tenInchScreenshots/` for 10-inch tablet screenshots (optional)
- `featureGraphic.png` in `fastlane/metadata/android/en-US/` (optional)

Use clear screenshots that show lessons, practice, and analysis screens.
